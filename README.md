# CutongBG

CutongBG adalah script otomatisasi untuk menghapus background gambar secara batch menggunakan layanan Picsart Background Remover.
Script ini memudahkan proses remove background secara batch tanpa perlu melakukannya satu per satu.

## Fitur Utama
- Hapus background otomatis dari gambar (output PNG transparan)
- Opsi proses 1 gambar, multi gambar, folder, atau multi folder
- Proses batch untuk multiple gambar
- Antarmuka yang sederhana dan mudah digunakan
- Output selalu PNG (transparan)

# Gampang banget:

### 1. Download:
```bash
git clone https://github.com/fannyf123/CutongBG.git
cd CutongBG
```
### (Gak ngerti git? Klik aja tombol ijo "Code" di atas terus "Download ZIP". Ekstrak dimana aja. Beres.)
### Atau klik langsung [GitHub Release](https://github.com/fannyf123/CutongBG/releases) terus download versi terbaru

### 2. Jalanin:
```bash
# Windows
Launcher.bat
```
(Tinggal double-klik file Launcher.bat. Ya udah gitu doang.)
(Tunggu proses unduh dependency selesai)

### 3. Pake:
   - Seret `gambar` / `multi gambar` / `folder` / `multi folder` ke aplikasi
   - Atau klik tab `Open File` / `Open Folder` untuk pilih yang mau dihapus backgroundnya
   - Klik tombol **Remove Background** dan tunggu hasilnya
   - Hasil tersimpan di folder `BG_REMOVED` di lokasi file asli

### Udah, cuma itu!
### Serius. Kalau langkah segampang ini aja masih bingung, mungkin remove background bukan masalah terbesar kamu.

> **Note:** Kalau masih bingung tanya di grup jangan japri.
